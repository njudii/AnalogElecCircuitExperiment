# 模电实验LaTeX模板

对于匡院物理方向的学生来说，``模电打字实验``和``近代排版实验``是两门比较鸡肋的课程。在课程负担较重的大三下还要应付实验报告是一件烦心事。这个模板旨在把匡院学生从ben无意义的实验报告中解放出来，有更多精力去钻研和思考实验本身。
